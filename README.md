# apps_moviles

Proyecto de Apps Móviles
========================

Esta es una app que gestiona una pequeña biblioteca personal, permitiendo añadir, buscar y ver detalles de libros y películas.

## ¿Qué hace la app?
- Permite ver una lista de libros y películas
- Puedes añadir, editar y borrar elementos
- Consulta información de Google Books y OMDb

## ¿Cómo ejecutar?
1. Clona el repositorio
2. Abre el proyecto en Android Studio
3. Espera a que se sincronicen las dependencias
4. Ejecuta en un emulador o dispositivo físico

## Estructura principal
- app/src/main/java/com/example/mylibrary: Código fuente principal
- app/src/main/res: Recursos (layouts, imágenes, strings)
- build.gradle.kts y archivos de configuración
